# 💻 05. HCI / UX / 감성공학

## 📖 HCI ( Human Computer Interaction )

### 📖 정의

HCI는 **사람과 컴퓨터(시스템) 사이의 상호작용을 연구하여 사용자가 시스템을 보다 편리하고 안전하게 사용할 수 있도록 하는 학문**이다.

현재는 컴퓨터뿐만 아니라 모바일, 서비스, 디지털 콘텐츠 등 다양한 시스템과 사람의 상호작용까지 포함하는 개념으로 확대되었다.

#### ✨ 특징

- 사용자 중심으로 시스템을 설계한다.
- 사용성과 효율성을 향상시키는 것을 목표로 한다.
- 사용자와 시스템의 상호작용을 연구한다.
- 최종 목표는 최적의 사용자 경험(UX)을 제공하는 것이다.

> 💡 HCI는 **좋은 시스템과 인터페이스를 설계하기 위한 기반 학문**이다.

## 😊 UX ( User Experience )

### 📖 정의

UX는 **사용자가 시스템이나 서비스를 이용하면서 느끼고 생각하게 되는 총체적인 경험과 만족**을 의미한다.

단순히 기능을 사용하는 경험뿐만 아니라 사용 과정에서 느끼는 감정, 편의성, 만족감까지 모두 포함한다.

#### ✨ 특징

- 사용자의 경험과 만족을 중요하게 고려한다.
- 기능뿐 아니라 감정과 심리까지 포함하는 개념이다.
- 사용자 중심 설계(User-Centered Design)의 핵심 요소이다.

### 📌 UX의 특징

| 특징 | 설명 |
|---|---|
| 주관성 | 개인의 성향과 인지 능력에 따라 경험이 달라진다. |
| 정황성 | 사용 환경과 상황에 영향을 받는다. |
| 총체성 | 사용 전·중·후의 모든 경험을 포함한다. |

<br/>

### 🔄 UI와 UX의 차이

| 구분 | UI | UX |
|---|---|---|
| 의미 | 사용자와 시스템의 인터페이스 | 사용자가 느끼는 전체 경험 |
| 초점 | 화면 구성과 사용성 | 경험과 만족도 |
| 목적 | 사용하기 쉽게 설계 | 사용하기 즐겁고 만족스럽게 설계 |

> 💡 **UI가 좋아도 UX가 반드시 좋은 것은 아니다.**  
> 좋은 UX를 위해서는 좋은 UI가 필요하다.

## ❤️ 감성공학 ( Kansei Engineering )

### 📖 정의

감성공학은 **인간의 감성과 심리적 특성을 제품이나 시스템 설계에 반영하는 공학 기술**이다.

인간의 감성을 과학적으로 분석하여 제품과 서비스의 만족도를 높이는 것을 목표로 한다.

#### ✨ 특징

- 인간의 감성과 심리를 제품 설계에 반영한다.
- HCI와 UX 설계에 활용된다.
- 인문·사회과학과 공학이 융합된 학문이다.
- 생체계측, 센서, 인공지능 등의 기술을 활용한다.

### 📌 감성공학의 요소 기술

| 요소 기술 | 설명 |
|---|---|
| 기반 기술 | 인간의 감성과 특성을 분석 |
| 구현 기술 | 감성을 반영한 인터페이스 구현 |
| 응용 기술 | 분석 결과를 제품과 서비스에 적용 |

## 📝 HCI / UX / 감성공학 핵심 정리

| 구분 | 핵심 내용 |
|---|---|
| HCI | 사용자와 시스템의 상호작용을 연구하는 학문 |
| UX | 사용자가 느끼는 전체 경험과 만족 |
| UI | 사용자와 시스템이 상호작용하는 인터페이스 |
| 감성공학 | 인간의 감성을 제품과 시스템 설계에 반영하는 기술 |
| UX 특징 | 주관성, 정황성, 총체성 |
| 감성공학 요소 기술 | 기반 기술, 구현 기술, 응용 기술 |
